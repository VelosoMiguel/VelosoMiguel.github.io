# Miguel Veloso — Personal Portfolio

Personal portfolio website built with HTML, CSS, and vanilla JavaScript. Dark cybersecurity-inspired theme.

🌐 **Live site:** [velosomiguel.github.io](https://velosomiguel.github.io)

## Built With

- HTML5
- CSS3 (custom properties, animations, grid)
- Vanilla JavaScript (scroll animations, custom cursor, typing effect)

## Features

- Dark terminal/cybersecurity aesthetic
- Custom animated cursor
- Typing animation in hero section
- Scroll fade-in animations
- Fully responsive
